# 🎬 IU Digital Cinema - APP Pelis FullStack

Aplicación profesional para la gestión de películas y series, desarrollada con una arquitectura limpia y separación estricta de responsabilidades (Frontend y Backend).

Este proyecto permite a un **Administrador**:
- Registrar nuevas producciones (Películas y Series).
- Gestionar catálogos de Géneros, Directores, Productoras y Tipos.
- Consultar, editar e información relacional.
- Visualización dinámica en el Frontend.

---

# 🚀 Tecnologías Utilizadas

### Backend
- **Node.js + Express**
- **MongoDB + Mongoose**
- **Clean Architecture** (Capas: Routes, Controllers, Services, Models)

### Frontend
- **ReactJS (Vite)**
- **Bootstrap 5** (Diseño responsivo y componentes premium)
- **Axios** (Comunicación con API REST)
- **React Router Dom**

---

# 📦 Estructura del Proyecto

El proyecto está dividido en dos carpetas principales para garantizar la mantenibilidad:

```text
Proyecto_IA/
├── backend/       # API REST (Puerto 3000)
└── frontend/      # Cliente React (Puerto 5173)
```

---

# ⚙️ Instalación y Configuración

1. **Backend:**
   - Entrar en `cd backend`
   - Crear archivo `.env` basado en `.env.template`
   - Instalar dependencias: `npm install`
   - Ejecutar: `npm run dev`

2. **Frontend:**
   - Entrar en `cd frontend`
   - Instalar dependencias: `npm install`
   - Ejecutar: `npm run dev`
